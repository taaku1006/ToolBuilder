上記のタスクを分析し、1つのPythonスクリプトで完結できるか、複数のステップに分解すべきかを判断してください。

## 分解が必要な条件
- 複数シートの生成が必要
- 統計処理と書式設定が両方必要
- 複数の出力ファイルが必要
- 500行以上のデータに対する複雑な変換
- 印刷設定やページ設定が必要
- SPC管理図などの高度な統計計算

## 分解が不要な条件
- 単純なフィルタ・ソート・集計
- 1シートへの出力のみ
- 基本的なデータ変換（型変換、欠損値処理等）

## 出力形式
JSON形式で返答してください。
```json
{
  "decompose": true,
  "reasoning": "判断理由",
  "subtasks": [
    {
      "id": 1,
      "title": "サブタスクのタイトル",
      "description": "詳細な説明（何を入力とし何を出力するか）",
      "depends_on": [],
      "expected_output": "出力ファイル名"
    }
  ]
}
```

## ルール
- サブタスクは最大{max_subtasks}個まで
- 各サブタスクは独立して実行可能なPythonスクリプトとして実装される
- 最初のサブタスクは環境変数 INPUT_FILE から元ファイルを読み込む
- 中間出力は環境変数 OUTPUT_DIR に保存する
- 後続のサブタスクは OUTPUT_DIR から前のステップの出力を読み込む
- 最後のサブタスクが最終出力を OUTPUT_DIR に生成する
- decompose が false の場合、subtasks は1つだけにし、そのdescriptionにタスク全体の説明を入れること
- JSON以外の出力は一切しないこと
