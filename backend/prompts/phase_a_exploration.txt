【結合セル・テンプレートシートの分析（重要）】
ファイルに結合セルが含まれる場合、またはテンプレート/フォーム系のシートがある場合は、
以下を必ず追加で調査・出力すること:

1. 全シートの結合セル範囲を一覧表示（範囲文字列と、そのセルの値）
2. テンプレートシート（「テンプレート」「フォーム」「Template」という名前を含む、
   または結合セルが8個以上のシート）については、
   行ごとに内容を詳細出力:
   - セクションヘッダー行の行番号と内容
   - データを書き込む空白セルの行番号と列座標
   - 結合範囲ごとに「左上セル座標 = 値 or (空)」を表示

```python
# 結合セル分析の参考コード
import openpyxl
from openpyxl.utils import get_column_letter

wb = openpyxl.load_workbook(input_file, data_only=True)
for sheet_name in wb.sheetnames:
    ws = wb[sheet_name]
    merges = list(ws.merged_cells.ranges)
    print(f"\n=== {sheet_name} ===")
    print(f"結合セル数: {len(merges)}")
    for mr in merges:
        val = ws.cell(mr.min_row, mr.min_col).value
        rng = f"{get_column_letter(mr.min_col)}{mr.min_row}:{get_column_letter(mr.max_col)}{mr.max_row}"
        print(f"  {rng} = {repr(val)}")
    # 内容のある行を全表示
    for row in ws.iter_rows():
        items = []
        for c in row:
            if c.value is not None:
                items.append(f"{get_column_letter(c.column)}{c.row}={repr(c.value)}")
        if items:
            print(f"  行{row[0].row}: {', '.join(items)}")
```

重要なルール:
- 調査結果は全て print() で標準出力に出力すること（ファイル出力ではない）
- 環境変数 INPUT_FILE から対象ファイルのパスを取得すること
- 環境変数 OUTPUT_DIR は使わなくてよい（この段階ではファイル出力不要）
- broad な try/except で全体を囲まないこと
